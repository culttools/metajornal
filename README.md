# METAJORNAL

**The news as the news.**

A retro-styled global news aggregator that crawls 300+ progressive news sources across 60+ countries in 20+ languages, translates everything to English, clusters stories by topic, and ranks by how many outlets cover the same event. Nokia 1997 aesthetic. Drudge Report energy.

## Two Ways to Run

### 1. Local Server (live mode)

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:8088 — crawls every 30 min, auto-refreshes, fully interactive.

### 2. GitHub Pages (static mode)

Push to GitHub, enable Pages, and the Action runs every 30 minutes:
1. Crawls all 300+ feeds
2. Translates non-English headlines
3. Clusters related stories
4. Exports `data.json`
5. Deploys `static/` to GitHub Pages

The frontend auto-detects which mode it's in.

## Deploy to GitHub Pages

```bash
# 1. Create repo
gh repo create metajornal --public --source=. --push

# 2. Enable Pages: Settings → Pages → Source: "GitHub Actions"
# (or use gh-pages branch deployed by the workflow)

# 3. Optionally add OpenAI key for better headlines:
#    Settings → Secrets → OPENAI_API_KEY

# 4. Trigger first build:
gh workflow run generate.yml
```

Site will be live at `https://yourusername.github.io/metajornal/`

## Configuration

| Variable | Default | Description |
|---|---|---|
| `OPENAI_API_KEY` | — | AI headline rewriting (optional) |
| `OPENAI_MODEL` | `gpt-4o-mini` | Model for headlines |
| `CRAWL_INTERVAL` | `30` | Minutes between crawls (local mode) |
| `PORT` | `8088` | Local server port |
| `MAX_CONCURRENT_FEEDS` | `50` | Parallel feed fetches |
| `FEED_TIMEOUT` | `15` | Feed timeout (seconds) |

## Customization

**Add/remove sources:** Edit `sources.py` — each entry is `(name, feed_url, website, language, country)`.

**Change look:** Edit `static/style.css` — all colors are CSS variables in `:root`.

**Change story count:** Set `TOP_STORIES_COUNT` in `config.py`.

**Clustering sensitivity:** Adjust `SIMILARITY_THRESHOLD` in `config.py` (0.0–1.0, lower = more aggressive merging).

## Architecture

```
static/            → Frontend (deployable to any static host)
  index.html       → Metajornal UI
  style.css        → Nokia 1997 aesthetic
  script.js        → Dual-mode: live API or static JSON
  data.json        → Generated by pipeline (gitignored)

app.py             → FastAPI live server
generate.py        → Static site generator (for CI/CD)
crawler.py         → Async RSS crawler (aiohttp + feedparser)
processor.py       → Translation + headline cleanup + TF-IDF clustering
database.py        → SQLite with WAL mode
sources.py         → 300+ curated global RSS feeds
config.py          → All configuration

.github/workflows/generate.yml → GitHub Actions: crawl → process → deploy
```

## Sustainability

- **Free hosting:** GitHub Pages (static files)
- **Free compute:** GitHub Actions (2,000 min/month free tier)
- **Free translation:** Google Translate via deep-translator
- **Free clustering:** scikit-learn TF-IDF (runs in CI)
- **Optional AI:** OpenAI for headline optimization (rule-based fallback is free)
- **Cron schedule:** Every 30 min = ~1,440 runs/month × ~10 min = well within free tier
